# Personal Portfolio Website

## Overview
A clean and responsive personal portfolio website built to showcase my skills, projects, and contact information. The focus was on simplicity, readability, and a structured layout.

## Features
- Responsive design for mobile and desktop
- Smooth navigation between sections
- Minimal and professional UI
- Projects and skills showcase

## Technologies Used
- HTML
- CSS
- JavaScript

## Design Approach
The website follows a minimal design system with consistent spacing, typography, and color usage to maintain a professional look.

## How to Run
1. Clone or download the repository
2. Open index.html in your browser

## Future Improvements
- Add animations and transitions
- Improve accessibility
- Integrate backend for contact form
